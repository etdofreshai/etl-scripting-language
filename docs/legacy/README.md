## Legacy design material

This directory holds documents and example programs imported from the
companion `etdofreshai/etl-scripting` repository when the two ETL repos
were merged into this one.

The material here predates the current direction (end-terminated blocks,
Python compiler-0 to C, autopilot wave workflow) and reflects an earlier
indentation-only design with a Rust bootstrap. Treat it as reference
only — examples under `examples/legacy/` are not expected to parse with
the current compiler-0.
